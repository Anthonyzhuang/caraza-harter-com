Data Files Used in Our Research

  1. The Flag Database
    • Retrieve From https://archive.ics.uci.edu/ml/datasets/Flags
    • Attribution
      1. Name: Name of the country
      2. continent: 1=N.America, 2=S.America, 3=Europe, 4=Africa, 4=Asia, 6=Oceania
      3. religion: 0=Catholic, 1=Other Christian, 2=Muslim, 3=Buddhist, 4=Hindu, 5=Ethnic, 6=Marxist, 7=Others
      4. Colors
        • colours: Number of different colours in the flag
        • red: 0 if red absent, 1 if red present in the flag
        • Same for green, blue, gold(also yellow), white, black, and orange (also brown)
        • mainhue: predominant colour in the flag (tie-breaks decided by taking the topmost hue, if that fails then the most central hue, and if that fails the leftmost hue)
      5. Symbols
        • bars: Number of vertical bars in the flag
        • stripes: Number of horizontal stripes in the flag
        • crosses: Number of (upright) crosses
        • sunstars: Number of sun or star symbols
        • crescent: 1 if a crescent moon symbol present, else 0
  2. The Country Database
    • Retrieve From: https://www.cia.gov/library/publications/the-world-factbook/index.html
    • Attribution
      1. Name: Name of the country
      2. Agriculture: Percentage of total Gross Domestic Product that comes from agriculture production.
      3. coast2area: the ratio of the coastline length over the total area
  3. PNG Files of Country Flags
    • Retrieve from: https://github.com/hjnilsson/country-flags
